# نقش پنهان (Hidden Role Manager)

اپلیکیشن مدیریت بازی‌های نقش مخفی (مثل مافیا/ورول‌ف) برای گرداننده‌ی بازی — کاملاً به زبان فارسی.

## امکانات
- ساخت قالب‌های دلخواه: هر تیم و نقشی که بخوای با نام و توضیح خودت
- تقسیم خودکار و تصادفی نقش‌ها بین بازیکن‌ها
- نمایش نقش با پاس دادن گوشی (لمس برای دیدن نقش)
- تایمر روز/شب با هشدار صدا و ویبره در پایان زمان
- رأی‌گیری روز و حذف بازیکن
- ذخیره‌ی تاریخچه‌ی بازی‌ها و قالب‌های نقش روی حافظه‌ی گوشی

## راه‌اندازی اولیه (مهم)

فایل باینری `gradle/wrapper/gradle-wrapper.jar` عمداً توی این پروژه نیست
(چون این فایل باید توسط خود Gradle واقعی تولید بشه، نه دستی).
قبل از اولین push، توی GitHub Codespace این دستور رو بزن:

```bash
gradle wrapper --gradle-version 8.7 --distribution-type all
```

این دستور فایل‌های `gradlew`, `gradlew.bat`, `gradle/wrapper/gradle-wrapper.jar`
و `gradle/wrapper/gradle-wrapper.properties` رو می‌سازه. بعدش کامیت و پوش کن.
